The Pre-Execution Loop

Three deterministic steps to secure agent operations.

1. Agent asks

Before interacting with a target contract, the bot submits the address to the Sentinel Alpha API.

2. Sentinel scores

Our nodes analyze the contract, calculate a risk score, and return a cryptographic attestation.

3. Bot decides

The agent reads the machine-formatted action (PROCEED, REVIEW, BLOCK) and enforces policy.

  -H "Content-Type: application/json" \

import { createSentinelClient } from "@beezshield/sentinel";

const client = createSentinelClient({
  x402Payment: "tx:YOUR_TX_HASH"
});

const decision = await client.decideBeforeExecution({
  contract_address: "0x1111111111111111111111111111111111111111",
  chain: "base",
});

if (!decision.shouldExecute) {
  agent.halt("Unsafe contract detected.");
}
