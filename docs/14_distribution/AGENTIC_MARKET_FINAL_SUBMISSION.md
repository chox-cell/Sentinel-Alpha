## Full Description
Sentinel Alpha provides a production API for machine-enforced risk decisions before execution. It returns `score`, `confidence`, `action`, `emergency_signal`, and `attestation` in a single response shape for deterministic integration into trading bots, DeFi agents, autonomous wallets, and agent-builder workflows. Sentinel Alpha runs on Base, supports x402 payments, and is registered on ERC-8004/8004scan for discoverability and identity proof.
